## Requirements
* Python3
* Virtualenvironment

## To test the application:
1. clone this repo

2. Create a virtual environment and activate it
```virtualenv venv```

3. Activate your virtual environment
```source venv/bin/activate```

4. Get into the project folder
```cd TDD_demo```

5. Run the tests
```pytest -m unittest```
